# Power BI Dashboard

This directory contains the Power BI report used to visualize credit risk and loan portfolio characteristics based on the processed LendingClub data.

## File
- `credit-risk-portfolio.pbix`

## Purpose
The Power BI report provides an interactive view of key credit risk metrics, including default behavior across loan grades, terms, and borrower profiles.

## Usage Notes
The `.pbix` file is included for reference and reproducibility.  
For a quick overview of the results, please refer to the dashboard preview image in the main README.

The report is intended for technical review rather than direct viewing by non-technical audiences.
